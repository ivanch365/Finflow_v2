# **Finflow**
Personal finance management software for macOS and iOS.

## **What the Application Does**
Finflow is a personal financial management tool built natively for macOS and iPhone. It allows users to input, track, categorize, and visualize their financial activity across multiple accounts. The application lets users record deposits and payments, specifying the description, category, date, and amount for each transaction.
Users can assign each transaction to a category, manage multiple accounts simultaneously, and generate visual reports including pie charts and monthly summaries to better understand their spending habits.

## **Who Will Use the Application**
Finflow is designed for anyone who wants to keep better track of their finances and better understand their financial behaviour. It is particularly useful for:

* **Students** managing a limited budget
* **Young entrepreneurs** tracking business and personal expenses
* **Anyone new to budgeting** who wants a clean, simple starting point

The interface is designed to make monitoring cash flow intuitive, helping users identify spending habits and make informed financial decisions.

## **Why This Project Is of Interest**
This project was built out of a belief that financial literacy matters, especially at a young age. An application like Finflow makes it easier to track a budget, gain insight into spending patterns, and plan for the future — without the complexity of enterprise finance software.

# **Features**
## **Accounts**
* Create and manage **multiple accounts**, each with its own currency
* View **account balance** at a glance, updated in real time
* See **total balance across all accounts**, converted to any currency using live exchange rates
* Save individual accounts to a *.finacct* file and reload them later
* Delete accounts with a confirmation prompt

## **Transactions**
* Add **payments** and **deposits** with description, amount, category, and date
* View a full **transaction history** per account, sorted by date
* **Filter transactions** by a custom date range
* **Swipe to delete** on iOS, or use the inline trash button on macOS
* **Multi-select mode** to delete multiple transactions at once

## **Categories**
* Default categories included for both expenses and income
* **Add custom categories** with a name and emoji
* **Delete categories** you no longer need
* Categories are shared across all accounts

## **Charts and Reports**
* **Pie chart** showing spending breakdown by category (1-month or 1-year view)
* **Monthly report** showing income, expenses, net balance, category breakdown, and full transaction list
* Reports available per account or across all accounts combined

## **Currency and Exchange Rates**
* Each account can use a **different currency**
* **Live exchange rates** fetched automatically on launch
* View any account balance **converted to any supported currency**
* Total balance across accounts converted to a **single chosen currency**

## **Save and Load**
* Save any account to a *.finacct* file stored in your Documents folder
* **Open a saved file** to restore an account and all its transactions
* Data is stored **fully locally** — nothing is sent to any server
